# Assets Folder

Tato složka obsahuje obrázky a media soubory pro webovou stránku KDU-ČSL.

## Potřebné obrázky

Nahrajte následující obrázky do této složky:

### Logo a branding
- `logo.png` - Logo KDU-ČSL (doporučená velikost: 200x200px)

### Hero sekce
- `hero-image.jpg` - Hlavní obrázek pro úvodní sekci (doporučená velikost: 800x600px)

### O nás sekce
- `about-image.jpg` - Obrázek pro sekci O nás (doporučená velikost: 600x400px)

### Historie sekce
- `history-1919.jpg` - Založení strany (1919)
- `history-1938.jpg` - První republika
- `history-1945.jpg` - Odboj a exilová vláda
- `history-1948.jpg` - Poválečné období
- `history-1989.jpg` - Sametová revoluce
- `history-modern.jpg` - Současnost

### Lidé sekce
- `predseda.jpg` - Foto Mgr. Marka Výborného (doporučená velikost: 400x400px)
- `person-1.jpg` - Člen předsednictva 1
- `person-2.jpg` - Člen předsednictva 2
- `person-3.jpg` - Člen předsednictva 3
- `person-4.jpg` - Člen předsednictva 4

### Aktuality
- `news-1.jpg` - Obrázek pro aktualitu 1 (doporučená velikost: 600x400px)
- `news-2.jpg` - Obrázek pro aktualitu 2
- `news-3.jpg` - Obrázek pro aktualitu 3

## Formát obrázků

- **Logo**: PNG s průhledným pozadím
- **Fotografie**: JPG (optimalizované pro web, 100-200 KB)
- **Rozměry**: Zachovejte doporučené rozměry pro nejlepší zobrazení

## Zdroje

- Oficiální logo a grafický manuál: https://kdu.cz
- Historické fotografie: Archiv KDU-ČSL
- Aktuální fotografie: Oficiální web a tiskové materiály

## Placeholder obrázky

Pro testování můžete použít placeholder služby jako:
- https://placeholder.com/
- https://picsum.photos/

Příklad: `https://picsum.photos/800/600` pro hero obrázek
